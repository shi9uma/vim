## submodule

1.   `git submodule init`
2.   `git submodule add https://github.com/ma6174/vim.git submodule/ma6174`
3.   有更新：`git submodule update --remote submodule/ma6174`
4.   `git commit -am 'update submodule' && git push` 不会推送到子模块的 repo
